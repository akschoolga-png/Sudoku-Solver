SUDOKU SOLVER

HOW TO USE (will add an interactive version later)
  - clone reposity
  - create solution object and add empty sudoku
  - if needed add return or print board

HOW IT WORKS
  - scans board for empty space
  - uses backtracking to find solotions
  - checks look-up table
  - recursion to move to next availible space

DETAILS
  - Time Complexity O(9^m)
  - Space Complexity O(m)
  - Python 3 required
